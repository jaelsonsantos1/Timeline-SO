Timeline Sistemas Operacionais

1949

O computador EDSAC, construído pelo Cambridge University Mathematical Laboratory, é um dos primeiros a usar um sistema operacional, chamado de "Monitor".

1956

O IBM 704 é lançado com o IBSYS, um dos primeiros sistemas operacionais de grande porte.

1961

O sistema operacional CTSS (Compatible Time-Sharing System) é desenvolvido no MIT e permite que várias pessoas usem um computador ao mesmo tempo.

1964

O sistema operacional OS/360 é lançado pela IBM e se torna o principal sistema operacional de mainframes por muitos anos.

1970

O sistema operacional Unix é desenvolvido no Bell Labs.

1981

A Microsoft lança o MS-DOS para o IBM PC, que se torna um grande sucesso e um dos sistemas operacionais mais utilizados por muitos anos.

1985

A Microsoft lança o Windows 1.0, a primeira versão do sistema operacional gráfico para PCs.

1991

Linus Torvalds desenvolve o kernel do Linux, que se torna um sistema operacional popular para servidores e estações de trabalho.

2001

A Apple lança o Mac OS X, baseado no kernel do Unix e com uma interface gráfica moderna.

2001

A Microsoft lança o Windows XP, uma versão muito popular do sistema operacional que é utilizada por muitos anos.

2007

A Apple lançou o iOS para o iPhone, que apresentava uma interface de usuário totalmente nova para dispositivos móveis.

2008

O Google lançou o Android, um sistema operacional de código aberto para dispositivos móveis.

2015

A Microsoft lançou o Windows 10, que introduziu várias novas funcionalidades, incluindo o assistente virtual Cortana.
